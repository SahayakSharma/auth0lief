<h2>Staff/Care Worker</h2>

<b>Requirement 1 : </b> The staff should be able to clock in.

The staff members or the care workers can select a location from their dashboard (location registered by the manager) and clock in into the location if their current location is inside the distance range specified by the manager from the desired location. When clocking in, the staff can even provide an optional note. They are not allowed to clock in when outiside the perimeter.
